### dtmcs

This folder contains the models, mostly taken [from here](https://github.com/prismmodelchecker/prism-benchmarks/tree/master/models/dtmcs) with added labels, as well as the scripts used to generate and process the results.

`robot` contains the script used to generate the prism models for that scenario.

The compiled results of the experiments can be found in `results/fmcad` (all measurements are in ms).
